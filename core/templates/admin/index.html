lightbulb Admin Quick Start Guide

1. Build Foundation

Add Class Sections (e.g., Grade 10 - Rizal) and core Subjects. Combine them into active Class Offerings with assigned teachers.

2. Assign Roles

Create Student Accounts. Assign teachers to Teacher Advisories to grant them full access to their Adviser Portal.

3. Manage Enrollments

Enroll students individually or group class offerings into Enrollment Groups so advisers can bulk enroll their students instantly.

4. Initialize Grading

Create active Grading Periods (e.g., Q1) and set DepEd Grade Weight Configs (WW/PT/QA) so teachers can start grading.

info Pro Tip: Save time by using the SIS Import Tools to bulk upload users, subjects, and enrollments from CSV files.
